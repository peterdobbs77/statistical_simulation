Matlab function randn()

clc;clear;close all;rng('default');
for n = [10^1,10^2,10^3,10^4,10^5,10^6]
% built-in Matlab
z = randn(n,1);
% Box-Muller Transform
u = rand(n,2);
z_u = sqrt(-2*log(u(:,1))).*sin(2*pi*u(:,2));
% compare histograms
figure
histogram(z,100); hold on;
histogram(z_u,100); hold off;
xlabel('Z');ylabel('Frequency');
legend('randn()','approx','location','best');
% compare mean and variance
fprintf('randn(), n=%i: mean = %i, var = %i\n',n,mean(z),var(z));
fprintf('approx, n=%i: mean = %i, var = %i\n',n,mean(z_u),var(z_u));
end

Wishart PDF

We can generate random matrix variate observations from the Wishart PDF via the Matlab function wishrnd(Sigma,nu).
p=2; % vector dimension
n=10; % sample size
m=10^6; % number of samples
nu=n-1; % degrees of freedom
% covariance matrix
Sigma = [2^2,2*4*.75; 2*4*.75,4^2];
A = chol(Sigma)';
% generate simulated observations
XX = A*randn(p,n*m)+repmat(mu,[1,n*m]); %*
  • Cheated, and used Matlab's generator. In future, need to generate these from .
X = zeros(p,n,m);
for j=1:m
X(:,:,j) = XX(:,(j-1)*n+1:n*j);
end
meanX = squeeze(mean(X,2));
covX = zeros(p,p,m);
for j=1:m
covX(:,:,j) = cov(squeeze(X(:,:,j)'));
end
% compute the sum of squares in each sample
Wmat = (n-1)*covX;
